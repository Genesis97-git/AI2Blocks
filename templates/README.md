# Templates

This directory contains reusable resources used during AI2Blocks development.

## apps/

Complete AI2Blocks example applications.

These demonstrate the language and serve as integration tests.

## snippets/

Small reusable AI2Blocks code fragments.

Useful for testing individual language features.

## reference-xml/

Raw Blockly XML exported from MIT App Inventor.

These files are used for reverse engineering and verifying generated XML.
